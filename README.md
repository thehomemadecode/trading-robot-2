The trading robot for
![Binance](https://img.shields.io/badge/Binance-FCD535?style=for-the-badge&logo=binance&logoColor=white)

## Installation
```bash
pip install -r requirements.txt &&
python3 setup.py build_ext --inplace
```

## Run
```bash
python3 trobot2.py
```
